Introduction

This exercise uses linear regression in R to predict value of equity trade for a day for a given volume of trade .

Data has been collected from National Stock Exchange , India.

Model

Linear Regression has been used to predict the value since the both Value and Volume are increasing in time.

Used bootstrap to calculate parameters of regression.

These parameters have been hardcoded due to coding issues with real time calculation.

Calculation

The trade value is calculated as :

**Where input$integer = 250000 (volume of trade)

require(bootstrap)
 c(1,2500000)%*%c(-21.5,0.00181)
